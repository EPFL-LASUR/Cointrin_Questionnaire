# Contributing to Cointrin Questionnaire

Thank you for your interest in contributing to the Cointrin Questionnaire project!

## How to Contribute

### Reporting Issues
- Use the GitHub issue tracker to report bugs or suggest features
- Provide clear descriptions and steps to reproduce issues
- Include screenshots if relevant

### Making Changes

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes** following the guidelines below
4. **Test your changes** thoroughly
5. **Commit with clear messages**: `git commit -m "Add feature: description"`
6. **Push to your fork**: `git push origin feature/your-feature-name`
7. **Open a Pull Request** with a clear description

## Development Guidelines

### Code Style
- Follow R code style conventions
- Use meaningful variable names
- Add comments for complex logic
- Maintain bilingual support (French/English) in user-facing text

### Testing
- Test the Shiny app locally before submitting
- Verify that data collection works correctly
- Check that analysis scripts run without errors
- Test with example data provided

### Documentation
- Update README.md if adding new features
- Add comments to explain non-obvious code
- Update QUICKSTART.md if changing installation or usage steps

## Areas for Contribution

### Enhancements
- Additional question types or sections
- Improved data visualization in analysis.R
- Export functionality for different formats (Excel, JSON)
- Multi-language support beyond French/English
- Mobile-responsive design improvements

### Bug Fixes
- Report and fix any issues found
- Improve error handling
- Enhance data validation

### Documentation
- Improve existing documentation
- Add examples or tutorials
- Translate documentation to other languages

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Focus on the best outcome for the project
- Help create a welcoming environment

## Questions?

If you have questions about contributing, please open an issue with the "question" label.

Thank you for helping improve the Cointrin Questionnaire!
